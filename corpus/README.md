# corpus/ — RAG 원문 코퍼스

이 폴더에 감사기준서·내부회계관리제도 지침 등 **원문 발췌**를 `.md` 또는 `.txt`로 넣으면,
앱 시작 시 자동으로 문단 단위 청킹되어 RAG 검색 대상에 포함됩니다.

## 넣는 방법
- 한 파일 = 한 기준서(또는 지침). 파일명 예: `auditing_540.md`, `icfr_guide.txt`
- 문단은 빈 줄(`\n\n`)로 구분. 20자 미만·`#`로 시작하는 줄은 청킹에서 제외됩니다.
- 조항 번호를 문단 앞에 붙여 두면 검색·인용 정확도가 올라갑니다.

## 주의 (프로젝트 규칙)
- 기준서 조항 번호와 원문은 **사용자가 직접 대조·입력**합니다. 도구는 조항을 지어내지 않습니다.
- 룰북(`rulebook/*.yaml`)의 `evidence.verified: false` 13건은 이 코퍼스로 원문 대조 후
  `true`로 승격하는 것이 다음 과제입니다.

현재 이 폴더가 비어 있어도 앱은 **룰북 YAML 자체를 청킹**해 RAG가 동작합니다.
코퍼스 문서를 추가할수록 검색 근거가 풍부해집니다.
